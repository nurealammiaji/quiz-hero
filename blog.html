Difference in map, filter, forEach & find

"map" creates a new array by transforming each element, "filter" creates a new array by selecting elements that match a condition, "forEach" iterates over an array and performs an action for each element, and "find" returns the first element that matches a condition.

Difference in localStorage, sessionStorage & cookies

"localStorage" and "sessionStorage" are web storage APIs that store data locally in the user's browser, with "localStorage" persisting data across sessions and "sessionStorage" being limited to the current session. "Cookies", on the other hand, are small text files stored by the browser primarily used for sending data back to the server with subsequent requests.

What is Event Loop

Using the "Event Loop", JavaScript can efficiently handle asynchronous operations without blocking the main execution thread. This mechanism ensures that time-consuming tasks are processed in the background and that the program remains responsive to user interactions.

Purpose of quizTimer() function

The "quizTimer()" function is used to calculate the time required to answer the quiz questions.

Purpose of chooseQuiz() function

The "chooseQuiz()" function is used to select, display and check the correctness of answers to quiz questions.
